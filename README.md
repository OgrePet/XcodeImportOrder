# XcodeImportOrder

Small and simple Xcode Extension to sort objective C #import directives

## Usage
### 1. Enable Cleanup
Check <kbd>System Preferences</kbd> -> <kbd>Extensions</kbd> -> <kbd>Xocde Source Editor</kbd> -> <kbd>Cleanup</kbd>
